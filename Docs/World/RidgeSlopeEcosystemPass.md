# Ridge Slope Ecosystem Pass

Created: 2026-06-25

## Goal

Make ridges, spurs, saddles, and slope breaks readable movement/orientation structures.

## Layers

- ridge crest affordance;
- spur shoulder route;
- saddle decision point;
- slope break transition;
- exposed slope vegetation;
- sheltered slope vegetation;
- sky-window opening;
- fatigue slope band.

## Behavior

Ridge travel can help orientation and movement, but it should not be universally safe. Steep or wet slopes should create time cost, fatigue, noise, and risk.

## Evidence

Review ridge crest, spur shoulder, saddle, slope break, exposed slope, sheltered slope, and sky-window point.

## Acceptance

A player can use landform and canopy openings to infer direction without map, compass, or route marker.
