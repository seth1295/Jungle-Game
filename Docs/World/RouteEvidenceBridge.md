# Route Evidence Bridge

Created: 2026-06-25

Route evidence connects route class, terrain channel, biome output, and evidence view.

Current route families:

- creek follow;
- ridge walk;
- spur descent;
- saddle cross;
- basin cross;
- coast move;
- lowland push;
- gully squeeze;
- slope traverse;
- canopy gap orient.

Each route family should eventually have a sample point, debug data, and repeatable visual evidence.
